# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-29

### Added
- Initial release of Hash Table Simulator
- GUI interface with tkinter
- Console-based interface
- Four collision handling modes:
  - Chaining with linked lists
  - Linear probing
  - Quadratic probing
  - Double hashing
- Real-time pseudocode visualization
- Step-by-step execution with animations
- Collision steps tracking
- Load factor monitoring
- Interactive operations (insert, search, delete, resize)
- Comprehensive demo examples
- Educational documentation

### Features
- Visual bucket representation with color coding
- Animated collision resolution
- Variable tracking display
- Auto-run mode with adjustable speed
- Tombstone support for deletions
- String key hashing
- Statistics and insights panel

### Documentation
- Complete README with usage guide
- Quick start guide
- Deployment instructions
- API documentation in code

### Testing
- Comprehensive test suite
- Demo examples for all modes
- Smoke tests for edge cases
